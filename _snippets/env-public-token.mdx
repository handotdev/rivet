<Info>
    In development, `RIVET_PUBLIC_TOKEN` will use the development token generated by `rivet init`.

    In production, `RIVET_PUBLIC_TOKEN` will be `undefined` and will use [tokenless authentication](/general/concepts/tokenless-authentication-web). This can be manually specified if you wish.

    More info:
    - [Handling game tokens](/general/concepts/handling-game-tokens)
    - [Development tokens](/general/concepts/dev-tokens)
    - [Public tokens](/general/concepts/token-types#namespace-public)
</Info>
